# project/archive/

Здесь хранятся архивные материалы проекта.

## Содержимое

| Папка | Что хранится |
|---|---|
| `template-history/` | История разработки самого шаблона Vibe-coding-docs (до публикации) |

## Для новых проектов

Если ты клонировал этот шаблон для своего проекта — папка `template-history/` тебя не касается. Это история создания шаблона, не твоего проекта.

Свои архивные материалы (устаревшие ТЗ, старые версии планов) можно класть сюда в отдельную папку, например `project/archive/my-project-v1/`.
