## Цель

Одно предложение: что должно быть сделано.
Например: «Создать форму регистрации пользователя с валидацией email».

## Шаги

1. Создать компонент `RegisterForm`
2. Добавить поля: email, пароль, подтверждение пароля
3. Валидация на фронте: формат email, минимум 8 символов для пароля
4. Подключить к Supabase Auth
5. Показать сообщение об успехе / ошибке

## Критерии готовности

- [ ] Форма отображается на странице `/register`
- [ ] Валидация работает до отправки на сервер
- [ ] Пользователь сохраняется в базе
- [ ] После регистрации — редирект на `/dashboard`
