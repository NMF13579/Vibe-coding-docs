# tasks/

Здесь хранятся файлы задач проекта.

## Формат файла задачи

Каждая задача — отдельный файл: `TASK-001.md`, `TASK-002.md` и т.д.

Используй шаблон из `docs/TASK-TEMPLATE.md`.

## Статусы

- `[ ]` — не начата
- `[~]` — в работе
- `[x]` — выполнена
- `[!]` — заблокирована

## Порядок работы

1. Создай файл `TASK-XXX.md` по шаблону
2. Опиши задачу и критерии готовности
3. Подтверди план с агентом ДО начала кода
4. После выполнения отметь статус и обнови `HANDOFF.md`
