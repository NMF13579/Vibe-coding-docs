# HANDOFF-SHORT — быстрое восстановление контекста

> Скопируй этот файл в чат с агентом в начале каждой сессии.  
> Заполни 5 полей — агент сразу знает где мы.  
> Полная история сессий → [HANDOFF.md](./HANDOFF.md)

---

## Шаблон (копировать целиком)

```
Восстанови контекст проекта:

1. Проект: [название]
2. Где остановились: [последнее действие / последняя задача]
3. Что сделано: [2-3 строки итогов прошлой сессии]
4. Следующий шаг: [что планировали делать дальше]
5. Активные риски/блокеры: [если есть — кратко, если нет — "нет"]

Прочитай HANDOFF.md и LAYER-3/project-status.md, подтверди что понял контекст.
```

---

## Пример заполненного шаблона

```
Восстанови контекст проекта:

1. Проект: Медицинская консультационная платформа
2. Где остановились: закончили TASK-003 — форма записи к врачу
3. Что сделано: реализована форма, валидация полей, интеграция с календарём
4. Следующий шаг: TASK-004 — уведомления пациенту по email/SMS
5. Активные риски/блокеры: нет API-ключа для SMS-провайдера

Прочитай HANDOFF.md и LAYER-3/project-status.md, подтверди что понял контекст.
```

---

## Когда использовать

| Ситуация | Действие |
|---|---|
| Новая сессия после перерыва | Вставить шаблон выше |
| Агент "забыл" что делали | Вставить шаблон выше |
| Смена IDE (Cursor → Claude Code) | Вставить шаблон выше |
| Полная потеря контекста | [LAYER-1/context-recovery.md](./LAYER-1/context-recovery.md) |
