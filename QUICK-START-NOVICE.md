## Для кого этот документ

**Для тебя**, если ты никогда не работал с AI-агентами или IDE.

**Что даст:** запустить первый проект без технического опыта.  
**Что НЕ даст:** настройку Cursor, установку зависимостей, написание кода.  
**Для кого НЕ он:** если ты уже работаешь в IDE → открой [QUICK-START.md](./QUICK-START.md).

После этого документа → вернись в [START.md](./START.md) и выбери следующий шаг.

# QUICK START — Для тех, кто не программист

> Если ты врач, дизайнер, менеджер или просто человек с идеей —  
> этот файл для тебя. Никакого терминала. Никакого кода.

---

## Что это вообще такое?

Представь: ты нанимаешь очень умного помощника.  
Но прежде чем он начнёт работать — ему нужна **папка с инструкциями**:  
как ты думаешь, что важно, как называть вещи, что можно, что нельзя.

Этот шаблон — и есть такая папка.  
Ты один раз настраиваешь правила, дальше ИИ-агент работает по ним сам.

## Сначала — выбери уровень проекта

| Уровень | Кому подходит | Время до первого результата |
|---|---|---|
| ⚡ **0 — Быстрый старт** | Хочу попробовать, идея ещё сырая | ~15 минут до первого кода |
| 🏗️ **1 — Стандартный MVP** | Есть идея, хочу структуру и план | ~45 минут до первого кода |
| 🚀 **2 — Production MVP** | Серьёзный продукт с реальными пользователями | ~1.5–2 часа до первого кода |

> Подробно о каждом уровне — в [`LAYER-1/levels-guide.md`](./LAYER-1/levels-guide.md)

## Шаг 1️⃣ — Скопировать шаблон к себе

1. Открой страницу шаблона на GitHub:  
   👉 **[github.com/NMF13579/Vibe-coding-docs](https://github.com/NMF13579/Vibe-coding-docs)**

2. Нажми зелёную кнопку **«Use this template»** → **«Create a new repository»**

3. Придумай название своего проекта (например: `moya-klinika` или `moy-proekt`)

4. Нажми **«Create repository»**

Готово — у тебя теперь своя копия шаблона. ✅

---

## Шаг 2️⃣ — Открыть в Cursor

> Cursor — это редактор кода со встроенным ИИ-агентом.  
> Бесплатно скачать: [cursor.com](https://cursor.com)

1. Скачай и установи Cursor (если ещё нет)
2. В Cursor: **File → Clone Git Repository**
3. Вставь адрес своего репозитория (скопируй из GitHub → кнопка **«Code»** → HTTPS)
4. Выбери папку куда сохранить → **Clone**

Cursor откроет проект. Слева увидишь в корне `START.md`, `README.md`, `llms.txt` и папки: `LAYER-1`, `LAYER-2`, `LAYER-3`. Для людей первый маршрут — `START.md`; для ИИ при новой сессии — `llms.txt`. ✅

---

## Шаг 3️⃣ — Первый разговор с агентом

1. В Cursor нажми **Ctrl+L** (или **Cmd+L** на Mac) — откроется чат с агентом
2. Убедись что выбран режим **Agent** (не Chat)
3. Скопируй и отправь эту фразу:

```
Начнём
```

Агент задаст тебе вопросы о проекте — просто отвечай на русском. ✅

## Что будет дальше

После интервью агент сам:
- Составит описание проекта
- Предложит с чего начать
- Будет помнить контекст между сессиями

**Если потерялся** → напиши агенту: `Восстанови контекст`  
**Если что-то пошло не так** → напиши агенту: `Всё сломалось`  
**Если не знаешь что писать** → открой [FAQ.md](./FAQ.md)

## Часто задаваемые вопросы

**Нужно ли знать программирование?**  
Нет. Ты описываешь что хочешь словами — агент делает остальное.

**Что если я ошибусь?**  
Всё сохраняется в Git. Напиши агенту `Всё сломалось` — он вернёт как было.

**Сколько это стоит?**  
Cursor бесплатен для старта. Шаблон бесплатен полностью.

**Можно использовать для медицинских проектов?**  
Да. Шаблон создан в том числе для медицинских IT-проектов.
