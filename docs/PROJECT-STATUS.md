## Статусы

- `not-started` — файл существует, данных нет
- `draft` — агент подготовил черновик
- `pending-approval` — черновик показан пользователю, ждём ответа
- `accepted` — пользователь подтвердил, файл — источник истины
- `outdated` — устарел после изменений стратегии или объёма

## Правило обновления

Агент меняет статус файла в этой таблице:
- при создании черновика → `draft`
- при показе пользователю → `pending-approval`
- после явного согласия пользователя → `accepted` + дата
- при смене стратегии/скоупа → `outdated`

> Не обновляй статус автоматически без подтверждения для файлов с `Needs approval: yes`.
