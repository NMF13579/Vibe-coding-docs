# AGENTS — роли агентов в проекте

> Описывает какие AI-агенты используются и как они взаимодействуют.

---

## Шаблон агента

```
### [Название агента]
- Инструмент: Claude / Cursor / Lovable / другой
- Роль: что делает
- Ограничения: что не делает
- Файлы: какие документы читает при старте
```

---

## Активные агенты

### Основной агент
- **Инструмент:** (заполнить)
- **Роль:** Планирование, написание кода, обновление документации
- **Ограничения:** Не деплоит без явного разрешения владельца
- **Файлы:** START.md, CLAUDE.md, HANDOFF.md, memory-bank/

---

## Правила взаимодействия

- Агент всегда показывает план перед выполнением
- Агент останавливается при появлении неопределённости
- Агент не принимает архитектурных решений без владельца
- Агент обновляет HANDOFF.md после каждой сессии
