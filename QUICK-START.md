# QUICK START — Начало за 2 минуты

> **Минимум вопросов. Максимум результата.**

---

## Шаг 1️⃣ — Скопировать два файла

```bash
# Скопировать в корень проекта
cp template-sync.js ./my-project/
cp setup.js ./my-project/
```

---

## Шаг 2️⃣ — Запустить мастер

```bash
cd ./my-project
node setup.js
```

**Мастер спросит:**
1. Где находится папка `vibe-coding-docs`? → Укажи путь
2. Добавить эти файлы? → Ответь `да` или `нет`

Дальше всё происходит автоматически ✨

---

## Шаг 3️⃣ — Готово

После синхронизации мастер подскажет что делать дальше:

```bash
git status              # Посмотреть что добавилось
git add .               # Добавить в коммит
git commit -m "feat: добавлены файлы из шаблона"
```

---

---

## Если что-то пошло не так

```bash
# Откатить изменения
git reset --hard HEAD~1

# Попробовать снова
node setup.js
```
