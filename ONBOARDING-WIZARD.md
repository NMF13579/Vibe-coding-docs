# 🎯 Мастер настройки за 5 минут (для врачей без опыта программирования)

---

## Шаг 1: Проверь, что у тебя есть

- [ ] Установлен Cursor или VS Code.
- [ ] Есть доступ к AI-сервису (аккаунт активен, вход выполнен).
- [ ] Есть папка проекта `Vibe-coding-docs` на компьютере.

---

## Шаг 2: Открой проект в IDE

### Вариант A: Cursor (GUI)

1. Открой Cursor.
2. Нажми `File` → `Open Folder`.
3. Выбери папку проекта.
4. Убедись, что слева видны `START.md`, `README.md`, `LAYER-1/`, `LAYER-2/`, `LAYER-3/`.

### Вариант B: Claude Code (GUI)

1. Открой интерфейс Claude Code.
2. Выбери `Open Project` или `Open Folder`.
3. Подключи папку проекта.
4. Проверь, что в рабочей области отображаются те же базовые файлы.

---

## Шаг 3: Ответь на 3 вопроса

После открытия проекта отправь агенту ответы на три вопроса:

1. Какова цель проекта на ближайший этап?
2. Будут ли использоваться персональные данные (ПДн)?
3. Нужны ли внешние интеграции (МИС, CRM, платёжные системы, API)?

На основе этих ответов агент сам настроит контекст и предложит безопасный следующий шаг.

---

## Шаг 4: Запусти агента

1. Нажми `Cmd/Ctrl + K`.
2. Вставь шаблон:

```md
Прочитай `llms.txt` и следуй только ему (поведение после загрузки — `LAYER-1/agent-rules.md`).
Моя цель: [впиши цель].
ПДн: [да/нет].
Интеграции: [перечисли или "не нужны"].
Скажи коротко, где мы находимся, предложи следующий шаг и дождись подтверждения.
```

3. Нажми **Run**.
4. Подтверди действие словом `да` или `выполняй`.
