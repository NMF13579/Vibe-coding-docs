# START — единый старт для шаблона

> Role: instruction
> Trigger: новая сессия, старт после копирования шаблона
> Read-time: ~2 min
> Next: llms.txt

## Это шаблон, а не живой продукт

Этот репозиторий — исходный каркас для копирования в новые проекты.
Пустые файлы в `LAYER-2/` и `LAYER-3/` на старте допустимы.

## Что читать сразу после этого файла

1. `llms.txt` — карта маршрутов (он сам указывает: сначала `START.md`, затем он)
2. `HANDOFF.md` — состояние самого шаблона
3. `LAYER-3/project-status.md` — статус пакета шаблона

## Главная операционная модель

Основной поток: `shared/` + `stages/` + `project/PROJECT.md`.

`LAYER-1/`, `LAYER-2/`, `LAYER-3/` — полная библиотека правил и заготовок.
При конфликте приоритетов использовать `shared/priority-order.md`.

## Обязательное правило

Plan -> Confirm -> Execute.
Без подтверждения владельца изменения не вносятся.
