# Инструменты репозитория

> Trigger: нужно скопировать шаблон в новый проект или подтянуть обновления шаблона  
> Read-time: ~3 min  
> Filled-by: user  
> Needs-approval: no  
> Next: [template-sync.md](./template-sync.md)

## setup.js

**Что делает:** копирует шаблон в новый проект, создаёт базовую структуру папок.  
**Когда запускать:** один раз при создании нового проекта из шаблона.  
**Как запустить:** из корня репозитория-шаблона:

```bash
node LAYER-1/tools/setup.js
```

## template-sync.js

**Что делает:** синхронизирует изменения из шаблона Vibe-coding-docs в существующий проект без перезаписи проектных файлов.  
**Когда запускать:** когда вышло обновление шаблона и хочешь получить улучшения.  
**Как запустить:**

```bash
node LAYER-1/tools/template-sync.js <папка-шаблона> <папка-проекта> [--dry-run]
```

**Подробности:** [template-sync.md](./template-sync.md)
