# LAYER-1 / archive

Эта директория содержит **временные оперативные документы**, которые отработали своё назначение и были архивированы.

## Что сюда попадает
- Одноразовые аудит-отчёты (`AUDIT-REPORT*.md`, `final-dev-audit-report.md`)
- Отработанные планы фиксов (`FIX-PLAN.md`)
- Устаревшие дубли документов (`HANDOFF-SHORT.md`)

## Что сюда НЕ попадает
- Устаревшие версии канонических документов → `LAYER-1/deprecated/`
- Актуальные документы любого типа

## Актуальные точки входа
- Аудит: `../audit.md`
- Текущее состояние: `../../LAYER-3/STATE.md`
- Governance: `../document-governance.md`
